# jhd-company-painting

Apresenta uma landing page feita pra uma empresa de pintura elétrica e indetificação. Com design
limpo e responsivo apresenta o serviço oferecido pela empresa de forma clara e objetiva.

# Tecnologias ultilizadas

- HTML
- CSS
- JavaScript
- Bootstraps icons

# Oque aprendi com esse projeto?

Com esse projeto aprendi muito sobre design responsivo, limpo, hierarquia de elementos
em um layout, além do uso de IntersectionObservers com JS.
